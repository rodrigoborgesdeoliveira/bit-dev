#### Component ID

The component ID is a unique component identifier that consists of 'scope name' and 'component name'.
When named properly, an ID reveals the component's purpose, context, and owner.

<Image maxWidth={400} src={componentId} />

Use the component ID to perform actions on components in your workspace or remote scope.

To learn more, see [Component ID](https://bit.dev/docs/components/structure/component-id).
